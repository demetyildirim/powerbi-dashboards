# Sales Performance Overview

## Overview
This dashboard provides a high-level overview of sales performance, focusing on revenue, profit, and regional distribution.  
It is designed for quick executive-level insights rather than deep operational analysis.

## Dataset
- Source: Kaggle â€“ E-commerce Orders and Sales Performance Dataset
- Granularity: Transaction-level sales data
- Main fields include:
  - Order Date
  - Product
  - Region
  - Sales
  - Profit
  - Quantity

## Key KPIs
- **Total Revenue**
- **Total Profit**
- **Profit Margin**
- **Total Quantity Sold**

## Visuals Included
- Revenue trend over time
- Revenue by product
- Revenue by region
- Revenue share by region

## Technical Details
- Data imported into SQL Server
- Power BI connected using Import mode
- Data cleaned and shaped with Power Query
- KPIs calculated using DAX measures
- Dashboard designed with a clean, minimal, business-oriented layout

## Notes
This dashboard is intentionally kept simple as an introductory portfolio project, focusing on clarity, correctness, and presentation quality rather than advanced modeling.
